## 🎯 عن المشروع

**كتب السلف داتابيز** هو مشروع مفتوح المصدر يهدف إلى توفير النصوص الكاملة للكتب الإسلامية التراثية بصيغة قواعد بيانات SQLite، قابلة للتحميل المباشر برمجيًا دون أي إعدادات إضافية.

يعتمد المشروع **بالكامل** على [مكتبة كيزانه](https://kizana.app) (Kizanah) كمصدر أصلي وحيد لجميع النصوص والبيانات، مع إعادة هيكلتها لتكون قابلة للاستخدام البرمجي السهل.

### المبادئ:
- ✅ **تحميل مباشر من الرابط — بسطر واحد في بايثون**
- ✅ **كل كتاب ملف SQLite منفصل** — حمّل ما تحتاجه فقط
- ✅ **لا مفاتيح API، لا تسجيل، لا مصادقة**
- ✅ **منظم بالفئات العلمية (40 تصنيفًا)**
- ✅ **فهرسة كاملة مع روابط مباشرة لكل كتاب**

## 🐍 الاستخدام في بايثون

### تحميل كتاب واحد — سطر واحد فقط:

```python
import sqlite3, urllib.request

# رابط الكتاب المطلوب (من INDEX.md)
book_url = "https://raw.githubusercontent.com/ivanshero400/كتب-السلف-داتابيز/main/كتب/العقيدة/1.sqlite"
urllib.request.urlretrieve(book_url, "book_1.sqlite")

# فتح الكتاب وقراءته
con = sqlite3.connect("book_1.sqlite")
text = con.execute("SELECT content FROM b1").fetchone()[0]
print(text[:500])  # أول ٥٠٠ حرف
```

### تصفّح الفهرس برمجيًا:

```python
import sqlite3

# تحميل الفهرس
urllib.request.urlretrieve(
    "https://raw.githubusercontent.com/ivanshero400/كتب-السلف-داتابيز/main/index.sqlite",
    "index.sqlite"
)

con = sqlite3.connect("index.sqlite")

# البحث عن كتب في العقيدة
books = con.execute("""
    SELECT b.book_id, b.book_name, a.author_name
    FROM book b
    JOIN author_book ab ON b.book_id = ab.book_id
    JOIN author a ON ab.author_id = a.author_id
    WHERE b.book_category = 1
    ORDER BY b.book_id
""").fetchall()

for book in books[:10]:
    print(f"[{book[0]}] {book[1]} — {book[2]}")
```

## ⚠️ ملاحظات

- **المصدر**: جميع النصوص والبيانات مأخوذة من مكتبة **كيزانه** (Kizanah) حرفيًا دون تعديل.
